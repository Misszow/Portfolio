Temperature Based Fan Speed Controller using Arduino

Project information

A Temperature Based Fan Speed Controller using Arduino is a system that utilizes an Arduino microcontroller and an LM35 temperature sensor to adjust the speed of a fan based on the ambient temperature. The system measures the temperature with the LM35 sensor and then modulates the voltage of a PWM (Pulse Width Modulation) signal to control the speed of a DC fan. This type of system is commonly employed in electronic equipment or home automation systems to maintain desired temperature levels and enhance energy efficiency. The sensed temperature in Celsius scale and the corresponding fan speed in percentage are concurrently displayed on an LCD panel, ensuring user-friendly interaction with the system.
